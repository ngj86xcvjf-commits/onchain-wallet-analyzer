# Onchain Wallet Analyzer

Starter project for Web3 / blockchain analysis.

Analyze wallet activity and generate CSV reports.

## Features
- Load wallet watchlist from CSV
- Generate timestamped CSV reports
- Simple CLI interface
- Easy to extend for on-chain data

## Usage

```bash
python3 src/analyzer.py --wallets data/wallets.csv
```
